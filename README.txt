# Gestor de Contraseñas Seguro

Este sistema de gestión de contraseñas permite a los usuarios de diferentes empresas visualizar, crear, editar y eliminar contraseñas de manera segura.


# Estructura de la Base de Datos

El sistema utiliza SQL Server como base de datos relacional para almacenar información sobre usuarios, contraseñas encriptadas y otros datos necesarios para el funcionamiento del sistema. Para obtener más detalles sobre las tablas y procedimientos, consulte el archivo BD.txt.


# Estandares y tecnologías aplicadas
- MVC (Modelo-Vista-Controlador): El proyecto sigue el patrón de diseño MVC para separar la lógica de la aplicacion (modelo), la presentación (vista) y el control (controlador), lo que facilita la organización y mantenimiento del código.
- C#: Lenguaje de programación utilizado para desarrollar la aplicación ( .NET 8.0.202).
- SqlServer: Sistema de gestión de bases de datos utilizado para almacenar los datos del sistema.
- Git: Sistema de control de versiones utilizado para el desarrollo colaborativo del proyecto.
- GitHub: Plataforma de alojamiento de código utilizado para almacenar y compartir el repositorio del proyecto.
- VisualStudio: Entorno de desarrollo integrado para escribir, depurar y compilar el software
